Install firebase tools

Set aliases
$ firebase use --add
Note:
This command allow to set an alias to the selected project
For ***qa, set alias qa
For ***staging, set alias staging
For ***prod, set alias prod

Deploy 
QA
$ npm run publish-qa

Staging
$ npm run publish-staging

Production
$ npm run publish
